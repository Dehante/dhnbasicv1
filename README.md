## Install
Cara install melalui git clone!

```bash
$ termux-setup-storage
(setelah itu tekan izinkan)
$ pkg update -y
$ pkg upgrade -y
$ pkg install git -y
$ git clone https://github.com/Dehante/dhnbasicv1
$ cd dhnbasicv1
$ npm i / bash install.sh
$ npm audit fix
$ npm start
```
